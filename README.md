# Android CRUD Skill Test - PT. Akasha Wira International

Aplikasi Android Native berbasis Kotlin untuk mendemonstrasikan kemampuan CRUD (Create, Read, Update, Delete) menggunakan integrasi API pihak ketiga.

## 🚀 Fitur Utama
* **Read**: Menampilkan daftar user dari JSONPlaceholder API menggunakan Retrofit.
* **Create**: Menambah user baru melalui FloatingActionButton dan Dialog.
* **Update**: Mengubah nama user yang sudah ada melalui list item click.
* **Delete**: Menghapus user dari daftar dengan konfirmasi AlertDialog.

## 🛠️ Tech Stack
* **Language**: Kotlin
* **Networking**: Retrofit & GSON Converter
* **Image Loading**: Glide
* **UI Components**: RecyclerView, CardView, FloatingActionButton, RelativeLayout, LinearLayout.

## 📂 Struktur Project
* `MainActivity.kt`: Logika utama aplikasi dan manajemen data lokal.
* `UserAdapter.kt`: Penanganan tampilan daftar dan interaksi item.
* `ApiService.kt`: Konfigurasi endpoint Retrofit.
* `User.kt`: Model data untuk mapping JSON.
